# MiMo TTS 语音合成 Skill

本 skill 提供小米 MiMo TTS 语音合成能力。核心特点：**用自然语言描述你想要的声音效果**，模型会自动匹配情感、语速、音调、方言、非语言声音等。

## Agent 引导流程

当用户请求生成语音时，agent 应：

1. **询问声音效果** — 不要直接生成，先问用户想要什么感觉的声音
2. **根据用户描述生成提示词** — 将用户需求转换为 MiMo 情绪标记
3. **调用 TTS** — 使用优化后的命令生成音频

### 快速开始

```bash
node scripts/tts.js --text "（情绪描述）你的文本" --voice default_zh --output /tmp/output.mp3
```

## 参数说明

### 必需参数

| 参数     | 简写 | 说明                   | 默认值 |
| -------- | ---- | ---------------------- | ------ |
| `--text` | `-t` | 要转换为语音的文本内容 | 无     |

### 可选参数

| 参数       | 简写 | 说明                        | 默认值            |
| ---------- | ---- | --------------------------- | ----------------- |
| `--model`  | `-m` | 使用的 TTS 模型             | `mimo-v2-tts`     |
| `--format` | `-f` | 输出音频格式 (`mp3`, `wav`) | `mp3`             |
| `--voice`  | 无   | 音色选择                    | `mimo_default`    |
| `--output` | `-o` | 输出文件路径                | `output.<format>` |

### 音色选项

| 音色值         | 说明         |
| -------------- | ------------ |
| `mimo_default` | 默认音色     |
| `default_zh`   | 中文默认音色 |
| `default_en`   | 英文默认音色 |

### 文本格式技巧

| 格式          | 效果               | 示例                 |
| ------------- | ------------------ | -------------------- |
| 大写 / 全大写 | 加重、强调         | `这真的是太离谱了！` |
| 重复字符      | 语气延长或情绪增强 | `你真的太过分了！！` |
| 省略号        | 犹豫、语气减弱     | `我也不知道……`       |
| 连字符        | 逐字强调           | `我 - 说 - 不 - 行`  |

### 上下文自动推断

模型会根据文本内容自动推断语气，无需额外标记：

| 文本特征       | 自动效果             |
| -------------- | -------------------- |
| 带问号 `？`    | 自动升调（疑问语气） |
| 带感叹号 `！`  | 自动加重情绪         |
| 愤怒或激动内容 | 自动加快语速、加重音 |
| 叙述性文本     | 自然叙述语气         |

### 完整示例

```bash
# 完整示例：生气 + 东北话
node scripts/tts.js --text "（生气，东北话，语速快）你可真行啊你！这事儿你也能干得出来？" --voice default_zh

# 完整示例：温柔 + 悄悄话 + 叹气
node scripts/tts.js --text "（温柔，悄悄话，带叹气）唉……你先睡吧，我一会儿就回来。" --voice default_zh

# 完整示例：紧张 + 结巴 + 呼吸急促
node scripts/tts.js --text "（紧张，结巴，呼吸急促）我……我真的不是故意的……（深呼吸）" --voice default_zh

# 完整示例：疲惫
node scripts/tts.js --text "（疲惫，有气无力）师傅……到地方了叫我一声……我先眯一会儿。" --voice default_zh

# 完整示例：寒冷 + 发抖
node scripts/tts.js --text "（寒冷，发抖）呼——呼——这大兴安岭的雪……简直能把人骨头冻透了……" --voice default_zh
```

## 参数说明

| 参数       | 简写 | 说明                             | 默认值          |
| ---------- | ---- | -------------------------------- | --------------- |
| `--text`   | `-t` | 要转换为语音的文本（带情绪标记） | 必需            |
| `--format` | `-f` | 输出格式 `mp3`/`wav`             | `mp3`           |
| `--voice`  | 无   | 音色选择                         | `default_zh`    |
| `--output` | `-o` | 输出文件路径                     | `output.<格式>` |

## 输出文件

脚本会在指定路径生成音频文件。若未指定 `--output`，默认输出到 `output.<格式>`。
